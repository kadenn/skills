# Security

Agent skills and hooks can influence tool use. Inspect this repository before installation and review hook definitions when your host asks you to trust them.

Report a vulnerability privately through GitHub Security Advisories for `kadenn/skills`. Do not include live credentials or private repository content in a public issue.

The Shipit hook is a lightweight defense-in-depth check. It cannot guarantee that a commit is free of secrets. Use repository secret scanning and a dedicated scanner for enforcement.
